# wtfAnime

## Features
Type in an anime that you like (or don't like) for recommendations.
Type in anything but an anime for an explicit alert.

## Technologies
- HTML
- CSS
- JS
- AniList API V2 (https://github.com/AniList/ApiV2-GraphQL-Docs)
